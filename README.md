# FitnessAPI

Backend API для платформы фитнес-тренировок и здоровья

Проект развивается как серверная часть для аутентификации, тренировочного контента, отслеживания прогресса, планов тренировок и блога о здоровье

## Стек

- Java 21
- Spring Boot 4
- Spring Security
- Spring Data JPA
- PostgreSQL
- Liquibase
- Gradle

## Реализовано

- регистрация пользователя
- вход пользователя
- выдача JWT access token
- ротация refresh token с хранением сессий
- CRUD API для видео
- валидация, локализованные ошибки и миграции БД

## В планах

- профиль пользователя и личный кабинет
- тренировки, упражнения и планы тренировок
- трекинг прогресса
- избранное и рейтинги
- блог о здоровье и питании
- фильтрация, пагинация и ролевая авторизация

## Запуск

1. Создать `.env` на основе `.env.example`
2. Заполнить значения переменных
3. Подгрузить переменные окружения в терминал или IDE
4. Запустить приложение

Шаблон переменных лежит в [.env.example](C:\Обучение\Программирование\Java\FitnessAPI\.env.example)

Важно: Spring Boot сам по себе не читает `.env` автоматически. Перед запуском переменные нужно экспортировать в окружение или подключить файл через конфигурацию IDE
